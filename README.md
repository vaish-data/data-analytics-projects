# Cellphone Sales & Customer Analytics Project

## Objective
This project focuses on analyzing cellphone sales data to uncover sales performance, seasonal trends, and customer behavior patterns. The analysis combines Python-based data preprocessing and statistical techniques with interactive Tableau dashboards to deliver actionable business insights.

## Tools Used
- Python (Pandas, NumPy, Matplotlib)
- Tableau
- Excel

## Key Analysis
- Identified peak sales periods indicating seasonal demand  
- Highlighted high-performing products and customer segments  
- Revealed patterns in customer purchasing behavior supporting business decisions
